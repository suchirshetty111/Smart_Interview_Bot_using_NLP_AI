# 🎯 Smart Interview Bot using NLP & AI

An AI-powered interview preparation platform that conducts mock interviews, evaluates candidate responses using Natural Language Processing (NLP), and provides real-time feedback with improvement suggestions. The system helps students and job seekers enhance their interview skills through personalized practice sessions.

---

## 🚀 Features

- 🤖 AI-powered mock interviews
- 🗣️ Voice and text-based interaction
- 📝 Technical and HR interview questions
- 📊 Real-time answer evaluation
- 💡 NLP-based feedback and scoring
- 📈 Performance analytics
- 🔄 Adaptive question generation
- 🌐 User-friendly web interface

## 🎯 How It Works

1. User selects interview type (HR / Technical).
2. AI asks interview questions.
3. User answers using text or voice.
4. NLP model analyzes the response.
5. AI scores the answer based on:
   - Relevance
   - Confidence
   - Grammar
   - Completeness
6. Personalized feedback is generated.
7. Performance report is displayed.
